## 🎯 Objetivos e Metas

Este projeto tem como objetivo desenvolver um sistema de recomendação baseado em conteúdo utilizando dados de cursos da Udemy.

O projeto está alinhado ao Objetivo de Desenvolvimento Sustentável 8 da ONU, especialmente à meta 8.6, que visa reduzir a proporção de jovens que não estão empregados, estudando ou em formação. A aplicação de técnicas de Processamento de Linguagem Natural e Similaridade de Cosseno permite identificar cursos com conteúdos semelhantes e gerar recomendações personalizadas.
